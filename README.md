# PETROLYS-APP
PETROLYS adalah aplikasi analisis data real-time dan komprehensif yang didedikasikan untuk mengevaluasi kualitas minyak mentah global. Dirancang untuk para profesional di sektor energi, keuangan, dan logistik, aplikasi ini menyediakan wawasan end-to-end mulai dari data pasar (harga) hingga klasifikasi teknis minyak
